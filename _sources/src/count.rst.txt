Count data
==========

**count** ()

    Count the number of rows of the main dataframe
    
**count_nulls** (``field``)

    Count the number of null values in a rows
    
**count_empty** (``field``)

    Returns a list of empty row indices
    
**count_zero** (``field``)

    List of row with 0 values
    
**count_unique** (``field``)

    Return the number of unique values in a column
